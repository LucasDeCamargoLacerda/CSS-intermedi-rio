🌐 Portfólio Intermediário (HTML + CSS)

Este projeto é um portfólio estático desenvolvido com HTML e CSS, com foco em boas práticas de estruturação, estilização moderna e organização de layout.

🚀 Objetivo

Evoluir do básico para um nível intermediário em Front-End, aplicando:

Estrutura semântica com HTML5

Estilização com CSS3

Layout com Flexbox
Uso de gradientes
Componentização de elementos
Organização visual estilo portfólio
